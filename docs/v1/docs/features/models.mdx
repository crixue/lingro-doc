## 多模型对比功能

**多模型对比**功能让您能够同时向多个 AI 模型提问，并实时查看不同模型的回答，帮助您快速对比各个模型的表现和输出质量。

### 使用方法

#### 1. 激活 @ 提及

在输入框中输入 `@` 符号，系统会自动弹出模型选择面板。

#### 2. 选择对比模型

- 在弹出的模型列表中，点击想要对比的模型（目前最多 3 个）

#### 3. 确认选择

- 按 `Tab` 键或点击"确认"按钮完成选择
- 按 `Esc` 键可取消操作
- 已选模型会以标签形式显示在输入框上方

#### 4. 发送消息

输入您的问题后，按 `Enter` 或点击发送按钮。系统会同时向所有选中的模型发送请求。

#### 5. 查看对比结果

- **自动轮播**：在模型生成回答时，系统每 3 秒自动切换显示不同模型的输出
- **手动切换**：点击模型图标或使用左右箭头按钮切换查看
- **全屏对比**：所有模型完成回答后，点击对比图标可打开全屏对比视图，并排查看所有回答

### 快捷键

| 按键 | 功能 |
|------|------|
| `@` | 激活模型选择 |
| `↑` `↓` | 在模型列表中上下移动 |
| `Enter` | 选择/取消选择当前高亮的模型 |
| `Tab` | 确认选择并关闭面板 |
| `Esc` | 取消并关闭面板 |

### 使用建议

- 对于复杂问题，建议选择不同类型的模型进行对比（如基础模型 vs 高级模型）
- 使用对比功能可以帮助您了解不同模型的特长和局限性
- 所有模型完成回答后，建议使用全屏对比视图进行详细比较

## 智能模型切换

系统会根据您的使用场景智能推荐最适合的模型，无缝衔接各类场景。当您需要使用知识库检索等工具功能时，若当前模型不支持，系统会自动推荐合适的检索模型，确保功能正常使用，无需手动切换。

### 如何使用多模型切换功能？

1. **选择模型**：在聊天页面的工作区/会话设置中，点击"**模型配置**"面板下的"**模型选择**"按钮，即可打开模型选择器
2. **浏览模型详情**：在弹出的模型选择器中，您可以看到每个模型的：
    - 名称和别名
    - 详细描述
    - 支持的最大上下文长度
    - **价格信息（输入/输出价格）**
    - **是否支持工具调用**
    - 是否为免费模型

3. **筛选模型**：可通过关键词搜索或使用过滤器（免费/付费、模型系列、是否支持工具）快速找到所需模型
4. **调整参数**：选择模型后，还可以在"聊天参数"中设置：
    - **创意度**（精确/平衡/创意）
    - **是否包含历史消息及数量**
    - **是否允许网络搜索**
    - **最大token输出限制**

### 便捷的用量统计

在 [用量](https://chat.i-lingro.com/#/global-settings/usage) 页面，您可以直观地查看：
- 当月各模型的token用量
- 输入/输出token分别的消耗情况
- 各模型的调用次数和费用
- 通过图表和表格两种形式展示用量数据
