# 灵格若网页插件
一款强大的浏览器扩展，帮助您轻松收集网页内容，快速构建专属知识库

灵格若网页助手专为 [灵格若(Lingro) 平台](https://chat.i-lingro.com) 设计，让您能够轻松收集网页内容，快速构建专属知识库，或直接将内容发送到 AI 聊天会话中。无论您是企业用户还是个人用户，都能通过这款插件显著提升工作效率，让 AI 真正理解并服务于您的业务需求。

## 核心功能

### 1. 智能网页内容提取

| 功能 | 说明 |
|------|------|
| **整页抓取** | 一键获取完整网页内容，自动转换为 Markdown 格式 |
| **智能清洗** | 智能过滤广告、导航栏、Cookie 弹窗等噪音内容 |
| **精准选区** | 支持选中页面特定内容进行提取，灵活满足不同场景需求 |
| **元数据保留** | 自动捕获页面标题、URL、最后修改时间、语言、网站图标等关键信息 |

### 2. 双模式发送

#### 知识库模式

- 批量收集多个网页内容
- 一键发送到指定知识库
- 支持全选/取消全选操作
- 自动同步到灵格若知识库系统

#### 聊天会话模式

- 实时发送网页内容到当前聊天会话
- 单选模式，精准传递
- 自动检测活跃聊天窗口
- 无缝集成到对话流程

### 3. 便捷操作方式

#### 右键菜单
在任意网页右键即可快速保存内容

#### 快捷键支持

| 快捷键 | 功能 | 模式 |
|--------|------|------|
| `Ctrl+Shift+U` / `⌘+Shift+U` | 获取并发送到聊天 | 聊天模式 |
| `Ctrl+Shift+S` / `⌘+Shift+S` | 获取整个页面 | 知识库模式 |
| `Ctrl+Shift+Z` / `⌘+Shift+Z` | 获取选中内容 | 通用 |

#### 可视化管理
弹窗界面查看、管理所有已保存内容

### 4. 灵活配置

- 自定义知识库详情页链接
- 智能识别聊天会话页面
- 支持多种页面格式

## 安装插件

下面以Chrome浏览器为例，介绍如何安装灵格若网页助手插件。

### 方式一：Chrome 网上应用店安装

1. **访问 Chrome 网上应用店**
   
   点击访问：[灵格若网页插件 - Chrome 网上应用店](https://chromewebstore.google.com/detail/%E7%81%B5%E6%A0%BC%E8%8B%A5%E7%BD%91%E9%A1%B5%E6%8F%92%E4%BB%B6/fpblgagnbocegdigifgbianicnapiikh)
   
   > 或在 Chrome 网上应用店搜索 "灵格若"

2. **点击"添加至 Chrome"按钮**
   
### 方式二：手动安装

如果您无法访问 Chrome 网上应用店，可以选择手动安装：

#### 步骤 1：下载插件包

- **本地下载**：<a href={normalizeImagePath('/web-extensions/lingro-web-extension-latest.zip')} download style={{color: 'var(--rp-c-brand)', textDecoration: 'underline', cursor: 'pointer'}}>点击下载最新版本插件包</a>

- **从 GitHub 下载**：[备用链接下载最新版本插件包](https://github.com/crixue/lingro-doc/releases/download/v1.0.0/lingro-web-extension-latest.zip)

#### 步骤 2：开启开发者模式

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 在页面右上角，打开 **"开发者模式"** 开关

<img src={normalizeImagePath('/chrome-extensions-developer-mode.png')} />

#### 步骤 3：加载扩展程序

将 下载好的插件`lingro-web-extension-latest.zip`拖拽到扩展程序页面，浏览器会自动解压并安装插件。

#### 步骤 4：验证安装

安装成功后，您应该能看到：
- ✅ 扩展程序列表中出现 "灵格若网页插件"
- ✅ 浏览器工具栏出现灵格若图标

## 开始使用

成功安装灵格若网页插件，开启智能知识管理之旅！

> 提示：首次使用时，请确保已登录 [灵格若平台](https://chat.i-lingro.com)。

### 聊天会话模式

#### 步骤 1：创建新的聊天会话

首先请在工作区中创建一个新的聊天会话

<img src={normalizeImagePath('/new-chat-session.png')} />

#### 步骤 2：获取网页内容

在您浏览的网页上，如果您希望将获取页面内容发送到聊天会话中，您可以：

- 右键选择"获取整个页面并发送"，或是使用快捷键 `Ctrl+Shift+U` / `⌘+Shift+U`实现快速发送
- 如您只需发送部分内容，可先选中网页内容，再右键选择"获取选中内容，或使用快捷键 `Ctrl+Shift+Z` / `⌘+Shift+Z`，然后在弹出的插件窗口中点击"发送到聊天会话"按钮

<img src={normalizeImagePath('/send-content-to-chat-session.png')} />

#### 步骤 3：开始与 AI 聊天

内容发送成功后，回到您之前在灵格若中打开的聊天会话，您就可以看到已上传了获取的网页内容，无需额外操作，您可以直接基于这些内容与 AI 进行对话。

<img src={normalizeImagePath('/chat-use-extension-content.png')} />

### 知识库模式

#### 步骤 1：打开一个知识库详情页面

首先请在灵格若平台【知识库中心】打开一个【知识库详情】页面，复制该知识库详情链接地址，准备将网页内容发送到该目标知识库中。

<img src={normalizeImagePath('/open-a-kg-page.png')} />

#### 步骤 2：获取网页内容

在您浏览的网页上，如果您希望将获取页面内容发送到聊天会话中，您可以：

- 右键选择"获取整个页面"，或是使用快捷键 `Ctrl+Shift+S` / `⌘+Shift+S`将页面内容暂时缓存到插件中
- 如您只需发送部分内容，可先选中网页内容，再右键选择"获取选中内容"，或使用快捷键 `Ctrl+Shift+Z` / `⌘+Shift+Z`将页面部分内容暂时缓存到插件中

您可以重复上述操作，获取多个网页内容，所有获取的内容都会显示在插件弹窗中。

#### 步骤 3：发送到知识库

1. 打开插件弹窗，选择**知识库**的发送方式，填写刚才复制的知识库详情页链接地址，保存配置。
2. 选择需要发送的内容，可以单选或全选，然后点击"发送到知识库"按钮。

<img src={normalizeImagePath('/web-extension-popup-kb.png')} />

#### 步骤 4：构建知识库索引

内容发送成功后，回到灵格若平台的知识库详情页面，您就可以看到已上传了获取的网页内容。接下来，您可以点击"立即构建索引"按钮，系统会自动处理并生成知识库索引，完成后即可在聊天中使用该知识库进行检索和问答。
